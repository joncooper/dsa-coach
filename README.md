# DSA Coach

A local-first data structures and algorithms interview course app. The course content is original and uses public topic coverage only as inspiration.

## Commands

```bash
npm run dev
npm run build
npm run test
npm run test:e2e
npm run validate:content
npm run verify:references
```

## What Is Included

- React + TypeScript + Vite app with MDX support.
- 13 course modules, 13 lessons, 70 guided Python problems, 140 runnable bonus drills, and 12 quizzes.
- Browser-local Python execution through Pyodide in a Web Worker.
- IndexedDB progress, notes, submissions, settings, JSON export, and JSON import.
- Content validation, unit tests, and Playwright smoke tests.
- Reference-solution verification for every visible and hidden test.
- Production service worker and manifest for offline/PWA-style reuse.

## Content Policy

Do not copy locked or paid LeetCode lesson text, screenshots, quizzes, prompts, solutions, or code templates into this repo. New lessons and problems should be original and validated with `npm run validate:content`.
